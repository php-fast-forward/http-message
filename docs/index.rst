Fast Forward HTTP Message
=========================

**Fast Forward HTTP Message** is a PHP library providing utility classes and implementations for working with PSR-7 HTTP Messages, focusing on immutability, strict typing, and developer ergonomics. It extends and complements `nyholm/psr7`, offering additional structures for payload management and convenient JSON and payload-centric responses.
