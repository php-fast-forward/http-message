Coverage and Quality Links
==========================

Useful quality signals for this package:

- `Coverage Report <https://php-fast-forward.github.io/http-message/coverage/>`_
- `GitHub Actions <https://github.com/php-fast-forward/http-message/actions>`_

How to Use Them
---------------

- Use the coverage report when you want to confirm which public behaviors are exercised by tests.
- Use the GitHub Actions page when you want to inspect the latest automated test runs for the repository.
